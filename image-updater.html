üöó Updating Car Images

  1. Prepare Images

    Take 3-5 high-quality photos of each vehicle (exterior, interior, dashboard)

    Recommended size: 1200x800 pixels | Format: JPG

  2. Name Your Files
    brand-model-year-1.jpg
    brand-model-year-2.jpg
    brand-model-year-3.jpg

    Example: toyota-camry-2022-1.jpg

  3. Upload to Folder

    Place all images in: public/images/car-inventory/

  4. Update Database

    Go to Supabase Dashboard ‚Üí Table Editor ‚Üí cars table

    Update the images field with:

    [
      "images/car-inventory/toyota-camry-2022-1.jpg",
      "images/car-inventory/toyota-camry-2022-2.jpg",
      "images/car-inventory/toyota-camry-2022-3.jpg"
    ]
üí° Pro Tip:

Use the SQL script file UPDATE_IMAGES.sql in the project root for bulk updates!

üë• Updating Staff Photos

  1. Prepare Photos

    Professional headshots of each team member

    Recommended size: 800x800 pixels | Format: JPG

  2. Name Your Files
    firstname-lastname.jpg

    Example: emmanuel-adeyemi.jpg

  3. Upload to Folder

    Place all photos in: public/images/staff-photos/

  4. Update Database

    Go to Supabase Dashboard ‚Üí Table Editor ‚Üí staff table

    Update the photo_url field with the path

‚ö†Ô∏è Important Notes

‚ö†Ô∏è File Naming:

File names are case-sensitive! Always use lowercase letters and hyphens (not spaces or underscores).

‚ö†Ô∏è File Paths:

Always use forward slashes (/) in paths, never backslashes (\).

‚úÖ Image Quality:
  • Use high resolution images (but compress to reasonable file sizes)
  • Keep file sizes under 2MB for cars, 1MB for staff
  • Ensure good lighting and clean backgrounds
  • Use consistent photo styles across similar items
